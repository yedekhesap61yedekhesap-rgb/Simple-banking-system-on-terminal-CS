# Easy Banking System on Terminal (C#)

A simple banking application developed with C# that runs on the terminal.

## Features

- Account creation
- User login system
- Deposit money
- Withdraw money
- Money transfer
- Balance checking
- Basic account operations

## Technologies Used

- C#
- .NET

## How to Run

1. Clone the project.
2. Open the project folder.
3. Run the application:

The `bank.txt` file you will see in the folder is where registered users and users who will log in are stored. IBAN number, full name, and balance amount are stored in this file.

I wish you a good experience using it.

! Please make sure to set the correct path of the `bank.txt` file in the code for the program to work properly!
